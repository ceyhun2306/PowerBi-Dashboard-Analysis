# PowerBi-Dashboard-Analysis

A collection of interactive dashboards built using Power BI, focusing on data cleaning, modeling, and visually effective storytelling.

---

## Project Overview
This repository contains Power BI dashboard projects developed from raw datasets. The workflow includes data preprocessing in Excel, data modeling in Power BI, and building interactive dashboards to support data-driven insights.

The main goal is to transform raw data into clear, visually appealing, and actionable dashboards.

---

## Workflow

Each project follows a structured process:

1. **Data Collection**  
   - Raw data is obtained in CSV format  

2. **Data Cleaning (Excel)**  
   - Data is preprocessed and cleaned in Excel  
   - Missing values and inconsistencies are handled  

3. **Data Modeling (Power BI)**  
   - Data is imported into Power BI  
   - Data types are corrected  
   - Relationships between tables are created and validated  

4. **DAX & Date Table Creation**  
   - Custom DAX measures are written  
   - A date table is created from scratch for time-based analysis  

5. **Dashboard Development**  
   - Interactive dashboards are built using various visuals  
   - Focus on clean, user-friendly, and non-cluttered design  
   - Each visual is carefully formatted for clarity  

6. **Advanced Features**  
   - Tooltips for deeper insights  
   - Bookmarks for interactivity  
   - Page navigation for better user experience
