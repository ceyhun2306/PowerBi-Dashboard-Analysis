# 💊 Dərman Satışı Analizi Dashboardu

## 📌 Layihə Haqqında
Bu layihə dərman satışlarının zaman, məhsul və müştəri aspektindən analizini təqdim edən interaktiv Power BI dashboardudur.

Dashboard biznesin əsas performans göstəricilərini izləməyə, trendləri anlamağa və daha düzgün qərarlar verməyə imkan yaradır.

---

## 🛠 İstifadə Olunan Alətlər
- Power BI  
- Microsoft Excel  

---

## 🔄 İş Axını (Workflow)
- Məlumatların toplanması  
- Data təmizlənməsi  
- Data modelləşdirmə  
- DAX ölçülərinin yaradılması  
- Dashboard dizaynı  
- İnteraktiv funksiyaların tətbiqi (filter, bookmark, tooltip)  

---

## ✅ Dashboard-un Üstünlükləri
- Satış performansını bir baxışda analiz etmək imkanı  
- Zaman üzrə trend və dəyişikliklərin aydın görünməsi  
- Müştəri davranışının analizi  
- Ən gəlirli məhsulların müəyyən edilməsi  
- Data əsaslı qərarverməni dəstəkləməsi  

---

## 🚀 Qazanılan Bacarıqlar
- Power BI ilə interaktiv dashboard qurulması  
- DAX funksiyalarının tətbiqi  
- Data modelləşdirmə  
- Analitik düşüncə  
- Vizual dizayn və UX yanaşması
